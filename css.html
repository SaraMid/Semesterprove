CSS

CSS = Cascading Style Sheets

  • CSS er et sprog, som beskriver udseende, og gør det muligt at adskille struktur og udseende.
  • Første version blev udviklet af Håkon Wium Lie (Hos CERN)
  • CSS blev adopteret af W3C (en interesseorganisation), som overtog standardiseringsarbejde.
  • Første reelle browser som understøttede CSS, var IE3
  • Netscape 4.0 understøttede kort efter CSS, men da det var blevet tilføjet meget forhastet, havde det mange fejl.
  • Standarden har siden udviklet sig meget, og alle store browsere understøtter nu standarden.


Cascading

- Et sæt regler, som definerer hvordan modstridende CSS deklareringer løses
- Specificity: Mere specifik har forrang mod mere generel
- Source order: Seneste deklareringer har forrang mod tidligere deklareringer. Eksempelvis har ID prioritet over class
- Specificity har forrang mod Source order
- Anvendes !important, ignorerer den source order, men bør undgås.

Float

  • Elementet "flyder" i det overordnede element
  • Angiv hvilken side elementet skal flyde til
  • Indhold i overordnet element flyder udenom
  • Bruges til placering af billeder i tekst og/eller placering af elementer ved siden af hinanden

Clear

  • Afgrænser hvilke sider der må røre andre elementer i det overordnede element
  • Kan være: left, right, both (hvis intet må være på begge sider) og none
  • Kan bruges til at ophæve float
