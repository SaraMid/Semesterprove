CSS

CSS = Cascading Style Sheets

  • CSS er et sprog, som beskriver udseende, og gør det muligt at adskille struktur og udseende.
  • Første version blev udviklet af Håkon Wium Lie (Hos CERN)
  • CSS blev adopteret af W3C (en interesseorganisation), som overtog standardiseringsarbejde.
  • Første reelle browser som understøttede CSS, var IE3
  • Netscape 4.0 understøttede kort efter CSS, men da det var blevet tilføjet meget forhastet, havde det mange fejl.
  • Standarden har siden udviklet sig meget, og alle store browsere understøtter nu standarden.


Cascading

- Et sæt regler, som definerer hvordan modstridende CSS deklareringer løses
- Specificity: Mere specifik har forrang mod mere generel
- Source order: Seneste deklareringer har forrang mod tidligere deklareringer. Eksempelvis har ID prioritet over class
- Specificity har forrang mod Source order
- Anvendes !important, ignorerer den source order, men bør undgås.

Struktur og opsætning

I CSS opsættes koden anderledes end i HTML. I HTML giver du dine semantiske tags som skal styles, classes eller IDs, som du herefter skal bruge som selector når du skal style i CSS'en.



Selector: Selectoren fortæller hvad vi gerne vil style.
Declaration start: Dette angiver vores område, hvor vi begynder at skrive selve CSS'en
Property: Har er områderne, som vi gerne vil ændre i og/eller skabe, dette kunne eksempelvis være font type, farve osv.
Property/value seperator: Dette angiver hvor afslutningen på linjen med elementer vi gerne vil arbejde med er.
Value: Her angives hvordan vi gerne vil ændre elementet, eksempelvis at vi gerne vil have baggrundsfarven til a være yellow som på billedet over.
Declaration seperator: Dette angives, ved afslutningen af den value vi har ændret.
Declaration end: Dette angiver afslutningen på CSS'en til dette område.

Float

  • Elementet "flyder" i det overordnede element
  • Angiv hvilken side elementet skal flyde til
  • Indhold i overordnet element flyder udenom
  • Bruges til placering af billeder i tekst og/eller placering af elementer ved siden af hinanden

Clear

  • Afgrænser hvilke sider der må røre andre elementer i det overordnede element
  • Kan være: left, right, both (hvis intet må være på begge sider) og none
  • Kan bruges til at ophæve float

CSS Box Model


Termet "box model", anvendes i CSS når der er tale om design og layout.
CSS box modellen kan essentielt beskrives som en boks uden om alle HTML elementer. Disse består af margin, border, padding og det egentlige content. Disse kan ses på billeder herover.

• Content  - Contentet af boksen, hvor texts og billeder er.
• Padding - Et transparent område omkring contentet.
• Border - En Ramme rundt om både content og transparent.
• Margin - Et transparent område uden om borderen.
